# BitRecon

A reconnaissance and information gathering framework for security professionals.

## Features

- **Subdomain Enumeration** — DNS brute-force, CT logs, passive sources
- **Port Scanning** — SYN/TCP/UDP with service detection
- **DNS Recon** — Full record dump and zone transfer
- **Whois Lookup** — Registrant data, name servers, dates
- **Tech Profiling** — Framework, CMS, server fingerprinting
- **Email Harvest** — Scrape emails from pages and headers
- **Full Scan** — Run all tools sequentially with live terminal output

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Framer Motion
- React Router
- Lucide React
